# Proyecto Frontend - Next.js

---

Correr el Proyecto

```
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev

```


---

Descripción

Aplicación frontend desarrollada con **Next.js**, enfocada en una arquitectura clara, escalable y mantenible.
Se implementa una separación de responsabilidades entre:

```

- Pages (rutas)
- View (UI por pantalla)
- Common (componentes reutilizables)

```


Esto permite mantener orden incluso cuando el diseño se construye de forma progresiva.


---

Arquitectura del Proyecto

```bash

app/
│
├── (pages)        # Definición de rutas
├── view/          # UI completa por pantalla
├── common/        # Componentes reutilizables
├── components/    # Componentes específicos


```
